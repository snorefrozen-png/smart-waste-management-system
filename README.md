# smart-waste-management-system
## Description
A web-based platform for reporting and managing waste issues such as illegal dumping and uncollected garbage. The system allows residents to report issues, administrators to assign collectors, and users to provide feedback after resolution.

## Features
- Report waste issues (illegal dumping, uncollected garbage)
- Admin dashboard to manage and assign collectors
- Status tracking of reported issues
- User feedback system after issue resolution
- Analytics dashboard using Chart.js

## Tech Stack
- Backend: PHP (PDO)
- Database: MySQL
- Frontend: HTML5, CSS3, Bootstrap 5
- Analytics: Chart.js

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/smart-waste-management-system.git
2. Move the project folder to your WAMP/XAMPP www directory
3. Start Apache and MySQL from WAMP/XAMPP
4. Open your browser and go to:

http://localhost/smart-waste-management-system

## Database Setup
1. Open phpMyAdmin
2. Create a new database (e.g., smart_waste)
3. Import the database.sql file included in this repository
4. Update database connection settings in the project if necessary
